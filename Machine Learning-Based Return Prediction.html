Analysis of Machine Learning Models for Stock Return Prediction¶

The results showcase the performance of multiple machine learning models (LSTM, Random Forest, XGBoost, and Walk-Forward Random Forest) in predicting stock returns for BWXT, CEG, NEE, and VRT. The evaluation metrics used are RMSE (Root Mean Squared Error) and R² (coefficient of determination).

1. LSTM Results¶

  • RMSE: The LSTM model achieves relatively low RMSE values for BWXT and NEE but struggles with VRT, indicating more significant prediction errors for that company.
  • R²: Negative R² values suggest that the model performs worse than a simple mean-based predictor for most companies. This highlights potential limitations of the LSTM model in capturing stock return patterns effectively.

2. Random Forest Results¶

  • RMSE: Random Forest produces competitive RMSE values, notably lower than LSTM for BWXT and CEG, suggesting better accuracy in return prediction for these stocks.
  • R²: While the R² values are still negative, they are less extreme than those for LSTM, indicating relatively better performance in explaining return variations.

3. XGBoost Results¶

  • RMSE: XGBoost performs well on NEE with the lowest RMSE but shows higher error for BWXT and CEG.
  • R²: The R² values indicate mixed performance, with significant underperformance for BWXT, suggesting challenges in generalizing across different stock patterns.

4. Walk-Forward Random Forest Results¶

  • RMSE: This approach achieves balanced RMSE values across all stocks, with a notable improvement for BWXT and NEE compared to the standalone Random Forest.
  • R²: While still negative, the values are closest to zero compared to the other models, showing promise for this method's applicability in dynamic financial environments.

Key Takeaways¶

Model Strengths¶

  • The Walk-Forward Random Forest shows the most consistent performance across all stocks, making it a promising choice for practical use.
  • XGBoost demonstrates potential for specific stocks like NEE, where it achieves the lowest RMSE.

Challenges¶

  • The LSTM model, despite its capacity for sequence modeling, struggles with predictive accuracy and generalization in this financial context.
  • Negative R² values across models indicate that further feature engineering or alternative predictors may be necessary to capture stock return dynamics effectively.

Future Steps¶

  • Investigate feature importance and refine input variables
  • Explore ensemble approaches combining model strengths
  • Incorporate market sentiment or macroeconomic indicators for enhanced predictions

import ta  # Technical Analysis library

# Function to compute technical indicators
def add_technical_indicators(df):
    for ticker in df.columns:
        # RSI
        df[f'{ticker}_RSI'] = ta.momentum.RSIIndicator(df[ticker], window=14).rsi()
        
        # MACD
        macd = ta.trend.MACD(df[ticker])
        df[f'{ticker}_MACD'] = macd.macd()
        df[f'{ticker}_MACD_diff'] = macd.macd_diff()
        df[f'{ticker}_MACD_signal'] = macd.macd_signal()
        
        # Bollinger Bands
        bb = ta.volatility.BollingerBands(df[ticker], window=20)
        df[f'{ticker}_BB_high'] = bb.bollinger_hband()
        df[f'{ticker}_BB_low'] = bb.bollinger_lband()
        
    return df

1.11 Walk-Forward Optimization¶
Walk-forward optimization involves retraining the model at each time step with the most recent data.


















In [27]:






def walk_forward_prediction(model_func, X, y, initial_train_size, retrain_interval):
    predictions = []
    test_indices = []
    
    for i in range(initial_train_size, len(X), retrain_interval):
        X_train_wf = X[:i]
        y_train_wf = y[:i]
        X_test_wf = X[i:i+retrain_interval]
        y_test_wf = y[i:i+retrain_interval]
        
        # Train the model
        model = model_func()
        model.fit(X_train_wf, y_train_wf)
        
        # Predict
        y_pred_wf = model.predict(X_test_wf)
        predictions.extend(y_pred_wf)
        test_indices.extend(y_test_wf.index)
        
    return predictions, test_indices

# Define model function for Random Forest
def rf_model_func():
    return RandomForestRegressor(n_estimators=100, random_state=42)

    results_wf_rf.loc[ticker] = [rmse, r2]

print("Walk-Forward Random Forest Results:")
